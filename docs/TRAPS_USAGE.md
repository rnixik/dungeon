# Как использовать ловушки (Traps)

## Настройка в редакторе карт Tiled

### 1. Создание шипов с таймером (автоматическая активация)

На слое **objects** создайте объект:

**Тип объекта (Class/Type):** `trap_spikes`
**Размер:** 32x32 (один тайл)

**Обязательные свойства (Properties):**
- `activator` (string): `timer`
- `period` (float): период активации в секундах (например `2.5`)

**⚠️ ВАЖНО:** Все числовые параметры должны иметь тип **float**, а не string!

**Опциональные свойства для кастомизации:**
- `warningTime` (float): длительность фазы предупреждения (по умолчанию `0.5` сек)
- `activeTime` (float): длительность активной фазы (урон) (по умолчанию `0.4` сек)
- `cooldownTime` (float): длительность восстановления (по умолчанию `1.6` сек)
- `damage` (float): урон (по умолчанию `18` HP)
- `phase` (float): смещение фазы для создания паттернов (по умолчанию `0.0`)

**Пример 1 - Базовые шипы:**
```
Name: spike_corridor_1
Type: trap_spikes
Width: 32, Height: 32
Properties:
  - activator = "timer"
  - period = 2.5
```

**Пример 2 - Быстрые опасные шипы:**
```
Name: spike_danger_1
Type: trap_spikes
Width: 32, Height: 32
Properties:
  - activator = "timer"
  - period = 1.5
  - warningTime = 0.3
  - activeTime = 0.6
  - damage = 30
```

Шипы будут активироваться автоматически с заданным периодом.

---

### 2. Создание шипов с триггером (активация при наступании)

#### Шаг 1: Создайте триггер

На слое **objects** создайте объект:

**Тип объекта:** `trigger`

**Свойства:**
- `name` (string): уникальный ID триггера, например `button_1`
- `target` (string): ID группы, которую активирует этот триггер, например `trap_group_1`

**Важно:** Нарисуйте прямоугольник в том месте, где игрок должен наступить.

**Пример:**
```
Name: button_entrance
Type: trigger
Width: 32
Height: 32
Properties:
  - target = "trap_group_entrance"
```

#### Шаг 2: Создайте шипы, связанные с триггером

На слое **objects** создайте объект:

**Тип объекта:** `trap_spikes`

**Свойства:**
- `name` (string): уникальный ID ловушки, например `spike_trap_1`
- `group` (string): ID группы (должен совпадать с `target` триггера), например `trap_group_1`

**Пример:**
```
Name: spike_trap_entrance_1
Type: trap_spikes
Properties:
  - group = "trap_group_entrance"
```

**Как работает:**
1. Игрок наступает на область триггера `button_entrance`
2. Триггер активирует все объекты с `group = "trap_group_entrance"`
3. Шипы `spike_trap_entrance_1` активируются через FSM

## Параметры ловушки

### Параметры по умолчанию:

```go
warningTime:  0.5   // Время предупреждения (телеграф) - 0.5 сек
activeTime:   0.4   // Время активной фазы (урон) - 0.4 сек
cooldownTime: 1.6   // Время восстановления - 1.6 сек
damage:       18    // Урон - 18 HP
```

### Кастомизация через properties в Tiled:

Вы можете переопределить любой параметр, добавив property к объекту:

| Property | Тип | Описание | По умолчанию |
|----------|-----|----------|--------------|
| `warningTime` | float | Длительность телеграфа и rising-анимации | 0.5 |
| `activeTime` | float | Длительность активной фазы (урон) | 0.4 |
| `cooldownTime` | float | Длительность восстановления | 1.6 |
| `damage` | float | Урон за попадание | 18 |
| `period` | float | Период активации (для timer) | 2.5 |
| `phase` | float | Смещение фазы (для timer) | 0.0 |

**Пример кастомных шипов:**
```
Type: trap_spikes
Properties:
  - activator = "timer"
  - period = 3.0
  - warningTime = 0.8    // Длинный телеграф
  - activeTime = 0.3     // Короткий урон
  - cooldownTime = 2.0   // Долгое восстановление
  - damage = 25          // Больше урона
```

## Состояния ловушки (FSM)

1. **Armed** (готова) - кадр 5, шипы спрятаны
2. **Warning** (предупреждение) - кадры 6-11:
   - Кадр 6: желтый телеграф (~0.1 сек)
   - Кадры 7-11: шипы поднимаются (~0.4 сек) - **урон уже наносится!**
3. **Active** (активна) - кадр 0, наносит урон
4. **Cooldown** (восстановление) - кадры 1-4, шипы опускаются

**⚠️ Важно:** Урон начинает наноситься с момента, когда шипы начинают подниматься (кадры 7-11), а не только на кадре 0!

**Общее время цикла:** 0.5 + 0.4 + 1.6 = 2.5 секунды

**Как работает урон:**
- Урон наносится **один раз** при пересечении с активной ловушкой
- Если игрок стоит на ловушке, урон наносится при активации
- Если игрок наступает на активную ловушку, урон наносится сразу
- Повторный урон возможен только после нового цикла активации

## Примеры паттернов

### Паттерн 1: Волна шипов (последовательная активация)

Создайте несколько шипов с одинаковым периодом, но разными фазами:

**⚠️ ВАЖНО:** Фаза должна быть **меньше** периода! Если фаза больше, она будет нормализована по модулю.

**Вариант 1 - Дробные фазы:**
```
spike_wave_1:
  - activator = "timer"
  - period = 1.0
  - phase = 0.0

spike_wave_2:
  - activator = "timer"
  - period = 1.0
  - phase = 0.25   // Через 0.25 сек после первого

spike_wave_3:
  - activator = "timer"
  - period = 1.0
  - phase = 0.5    // Через 0.5 сек после первого

spike_wave_4:
  - activator = "timer"
  - period = 1.0
  - phase = 0.75   // Через 0.75 сек после первого
```

**Вариант 2 - Большой период:**
```
spike_wave_1:
  - activator = "timer"
  - period = 4.0
  - phase = 0.0

spike_wave_2:
  - activator = "timer"
  - period = 4.0
  - phase = 1.0    // Через 1 сек после первого

spike_wave_3:
  - activator = "timer"
  - period = 4.0
  - phase = 2.0    // Через 2 сек после первого

spike_wave_4:
  - activator = "timer"
  - period = 4.0
  - phase = 3.0    // Через 3 сек после первого
```

Шипы будут активироваться по очереди с заданным интервалом.

---

### Паттерн 2: Быстрые и медленные шипы

```
// Быстрые опасные шипы
spike_fast:
  - activator = "timer"
  - period = 1.5
  - warningTime = 0.3
  - activeTime = 0.5
  - damage = 25

// Медленные безопасные шипы
spike_slow:
  - activator = "timer"
  - period = 4.0
  - warningTime = 1.0
  - activeTime = 0.3
  - damage = 15
```

---

### Паттерн 3: Шахматный паттерн

Разместите шипы в шахматном порядке с противофазами:

```
// Черные клетки
spike_chess_black_*:
  - activator = "timer"
  - period = 3.0
  - phase = 0.0

// Белые клетки
spike_chess_white_*:
  - activator = "timer"
  - period = 3.0
  - phase = 1.5
```

### Паттерн 2: Опасный коридор с кнопкой

```
button_safe: trigger с target="corridor_traps"
spike_corridor_1: group="corridor_traps"
spike_corridor_2: group="corridor_traps"
spike_corridor_3: group="corridor_traps"
```

Игрок должен нажать кнопку, чтобы активировать все шипы в коридоре.

### Паттерн 3: Постоянно активные шипы

```
spike_danger: activator=timer, period=2.0
```

Шипы будут активироваться каждые 2 секунды постоянно.

---

## Кадры анимации шипов

Убедитесь, что спрайтшит имеет 12 кадров:
- **Кадр 0:** Полностью выдвинутые шипы (active)
- **Кадры 1-4:** Опускающиеся шипы (cooldown)
- **Кадр 5:** Полностью спрятанные (armed)
- **Кадр 6:** Телеграф-индикатор (warning start)
- **Кадры 7-11:** Поднимающиеся шипы (warning->active)

---

## Debugging

Чтобы увидеть состояние ловушек, проверьте консоль сервера - при создании трапов выводится информация о каждой ловушке.
